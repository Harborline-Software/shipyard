## Notes

**Built 2026-05-01 across 8 phases.** P1: `Sunfish.Foundation.Transport` package scaffold + contracts (`TransportTier` enum / `IPeerTransport` / `IDuplexStream` / `ITransportSelector` / `IMeshVpnAdapter`) + value records (`PeerEndpoint` / `MeshNodeStatus` / `MeshPeer` / `MeshDeviceRegistration`); `PeerId` reused from `Sunfish.Federation.Common` per A1. P2: `DefaultTransportSelector` with A4-pinned per-tier budgets (T1: 2s, T2: 5s/adapter, T3: 10s; per-handshake 2s) + 30s selection-result cache + registration-order primary + lexicographic tie-break for mesh adapters + graceful fall-through on every failure mode. P3: `MdnsPeerTransport` Tier-1 backed by `Makaretu.Dns.Multicast.New`; TXT layout `peer` (PeerId base64url) + `port`; `ConnectAsync` opens real TCP via `TcpDuplexStream` adapter; LAN smoke test gated on `SUNFISH_MDNS_TESTS=1`. P4: `providers-mesh-headscale` first Tier-2 adapter (BSD-3 license posture; no Headscale-Sharp dep); `HeadscaleClient` REST shell with virtual methods for test substitution; `HeadscaleMeshAdapter` with short-TTL `IsAvailable` cache + `tag:sunfish-peer-{base64url}` ACL-tag mapping for the (DeviceId, PeerId) two-field shape per A1; `ConnectAsync` opens TCP at WG data-plane port (default 51820). P5: `BridgeRelayPeerTransport` Tier-3 always-tried fallback over `ClientWebSocket` → `WebSocketDuplexStream` adapter; ciphertext-only posture preserved per ADR 0031. P6: 5 new `AuditEventType` constants (`TransportTierSelected` / `MeshDeviceRegistered` / `MeshHandshakeCompleted` / `MeshTransportFailed` / `TransportFallbackToRelay`) + `TransportAuditPayloads` factory (alphabetized) + audit-enabled overloads on `DefaultTransportSelector` + `HeadscaleMeshAdapter` (W#32 both-or-neither pattern); selector emits on every cache-miss selection; cache-hit path skips emission. P7: `AddSunfishTransport()` + `AddSunfishTransport(TenantId)` DI extensions (audit-disabled / audit-enabled at the registration layer too) + `AddBridgeRelay(BridgeRelayOptions)` helper + `apps/docs/foundation/transport/{overview,headscale-setup}.md` operator guides + foundation toc entry. P8: this ledger flip. **87 tests across the foundation-transport + providers-mesh-headscale suites** (68 in foundation-transport: 9 P1 surface + 13 P2 selector + 11 P3 mDNS + 10 P5 BridgeRelay + 8 P6 selector audit + 7 P6 payload-shape + 10 P7 DI; 19 in providers-mesh-headscale: 16 P4 + 3 P6 adapter audit). All 5 hand-off halt-conditions stayed off the trip-wire. License posture clean (Headscale BSD-3; no SSPL/BSL deps). Tailscale/NetBird/WireGuard plain adapters deferred to follow-on workstreams; Bridge-hosted Headscale (Phase 2.3+) remains SPECULATIVE-PENDING-MEASUREMENT per A4. Substrate-only; consumer wiring (kernel-sync gossip daemon consuming `ITransportSelector`, native iOS NetworkExtension transport for W#23) is separate workstreams.
