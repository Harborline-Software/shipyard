# Sunfish.Blocks.Docs — Notices

The document-attachment model in this package is intellectually informed by the **Apache OFBiz** content module (Apache License 2.0) — specifically its separation of the content payload from cross-cluster references. No source code is copied.

The content-addressed-storage discipline (sha-256 dedup, immutable post-upload) draws on the broader literature around IPFS, Git, and content-addressable systems.
